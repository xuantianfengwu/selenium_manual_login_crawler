# Step1：执行dist/complete_example.exe文件
- 执行后，会有一个命令行界面启动，并弹出文件选择器。
- 选择包含要爬取公司列表的excel文件（eg：/static/complete_example/2022年融过资 表1.xlsx)

# Step2: 下载驱动，启动浏览器，等待登录
- 选择文件后，将会自动下载浏览器启动（chromedriver），之后启动chrome浏览器，并访问xunkebao.baidu.com。
- 程序会pending住，等待用户进行登录后手动触发下一步。

# Step3：检查产出文件，进入循环爬取
- 完成登陆后，在命令行输入回车，将会继续进行产出文件检测。
- 产出文件路径默认在/dist/static/output.xlsx。若文件不存在，将会新建；若文件存在，则会从中读取已爬取过信息的公司，确保不再重复爬取。

# Step4：循环爬取，随时退出
- 爬取过程中，每个公司的联系方式信息，会不断追加写入output.xlsx文件中。可随时关闭网页/关闭命令行窗口，实现中止爬取。
- 后续希望继续爬取，只需再次运行exe文件，并选取和前一次相同的excel文件。程序会自动根据output.xlsx中的内容，跳过已爬取过的公司，实现继续爬取的目的。
